Git & GitHub Tutorial

🚀 Getting Started

Set up your username and email for Git:

git config --global user.name "Your Name"
git config --global user.email "you@example.com"

📂 Initialize Repository

Create a project folder and initialize Git:

mkdir devops-git-project
cd devops-git-project
git init

📌 First Commit

Add a README file and commit:

echo "# My First Git Repo" > README.md
git add README.md
git commit -m "Initial commit with README"

🌐 Push to GitHub

Link your repo and push changes:

git remote add origin https://github.com/username/devops-git-project.git
git branch -M main
git push -u origin main
